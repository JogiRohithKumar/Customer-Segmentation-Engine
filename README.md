# 📊 Customer Segmentation Engine

Transform raw customer data into actionable business insights using Machine Learning.

### 🚀 Live Demo

https://customer-segmentation-engine-lo7qvs67rqmxk3sir5e6ey.streamlit.app/

---

## Problem

Businesses often have thousands of customers but treat them all the same.

This project automatically groups customers into meaningful segments based on purchasing behavior, helping businesses identify:

* High-value customers
* Budget-conscious customers
* Loyal customer groups
* Marketing opportunities

---

## Features

✅ Upload any customer CSV dataset

✅ Automatic data preview and validation

✅ Dynamic feature selection

✅ K-Means clustering

✅ Interactive visualizations

✅ Customer personas generation

✅ Business recommendations

✅ Download segmented dataset

---

## Example Business Use Case

A retail company wants to understand customer behavior.

Input:

| Annual Income | Spending Score |
| ------------- | -------------- |
| 15            | 39             |
| 81            | 93             |
| 48            | 45             |

Output:

| Segment           | Business Action          |
| ----------------- | ------------------------ |
| Premium Customers | Luxury product campaigns |
| Budget Customers  | Discount offers          |
| Regular Customers | Loyalty programs         |

Instead of analyzing thousands of records manually, the model automatically discovers customer groups.
